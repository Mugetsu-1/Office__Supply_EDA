# Office Supply Data Analysis

---

### Purpose
This repository contains a **practice project** created for building and showcasing skills in **Data Analysis (DA)** and **Data Science (DS)**. It is intended for learning and portfolio development on GitHub.

---

### Project Description
The project performs data analysis on an office supply dataset using Python and Jupyter Notebook. The focus is on data cleaning, exploratory data analysis (EDA), and visualization to gain practical experience with real-world-style data.

---

### Skills Demonstrated
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Data manipulation with Pandas  
- Data visualization with Matplotlib  
- Insight interpretation  

---

### Tools & Technologies

- Jupyter Notebook
